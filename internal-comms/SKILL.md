# Internal Comms — Case Study & Business Writing

## When to Use

After a project is completed and images are ready. This skill takes raw project information and produces publication-ready case study copy.

## Input Expected

- Project name
- Client name
- Industry / service type
- Project description (what was done)
- Challenge or problem (before state)
- Results or outcomes (after state)
- Timeframe
- Suburb / location
- Optional: client testimonial or quote

## Output

Formatted case study copy with:
- SEO-optimised headline
- Before → After structure
- Problem agitating paragraph
- Solution/process description
- Results with specifics (numbers where available)
- Client testimonial block (if provided)
- Call to action linking to services page

## Tone

- Professional but not corporate
- Specific, not generic
- Results-focused
- Written for the business's industry and local audience
